TableBank is a mobile application designed to digitize and simplify group savings management by replacing manual record-keeping with a secure and user-friendly system. Developed using Android Studio, Java, and XML, the current version implements a functional login interface with input validation and authentication, allowing users to access the system, add contributions, and view contribution history. The project focuses on improving transparency, accuracy, and accountability among members while establishing a scalable foundation for future features such as user registration, group management, real-time progress tracking, reporting, and database integration.
